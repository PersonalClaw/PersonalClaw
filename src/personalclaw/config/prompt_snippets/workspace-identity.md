A workspace is the working directory you operate in, with its own memory (preferences, projects, daily history). Different working directories have different memory — what you learn in one stays scoped to that directory.

Lessons have two scopes (use memory_remember tool to save):
- scope=global (default): shared across ALL workspaces. Use for universal preferences (e.g. 'always use dark mode').
- scope=workspace: only visible in this working directory. Use for project-specific rules (e.g. 'this repo uses pytest-asyncio strict mode').
